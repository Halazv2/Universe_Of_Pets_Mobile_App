# Universe_Of_Pets_Mobile_App

this is a mobile app for pets lovers, where they can find all the information about pets (check my other application for the web version pet adoption website [here](https://github.com/Halazv2/FindPet) )

## Features

- [x] Login
- [x] Register
- [x] Home
- [x] Profile
- [x] Search
- [x] add to cart
- [x] checkout
- [x] order history
- [x] order details

## Installation

1. Clone the repo
   ```sh
   git clone
    ```

2. Install NPM packages
    ```sh
    npm install
    ```
3. Install Pods
    ```sh
    cd ios
    pod install
    ```
4. Run the app
    ```sh
    npm run ios
    ```
    or
    ```sh
    npm run android
    ```
